# 📚 Important National & International Days  
### Competitive Exam Revision Website  
### By Arnav Batra  

A **sketchbook-themed responsive study website** made for quick revision of **important national and international days**, specially designed for **competitive exams** like:

- UPSC
- SSC
- Banking
- CDS
- NDA
- CUET
- Railways
- School GK Olympiads

The website includes **month-wise filtering, sticky notes for today's or upcoming observances, and a mobile-friendly notebook UI**.

---

## ✨ Features
- 📅 **Month-wise filter**
- 📌 **Sticky note widget**
  - shows **today’s important day**
  - otherwise displays **upcoming day**
- 📱 **Responsive on mobile and desktop**
- ✏️ **Sketchbook / notebook aesthetic**
- 🔗 Custom social buttons
- 📚 Major national + international exam dates included
- ⚡ Fast static website (HTML, CSS, JavaScript)
