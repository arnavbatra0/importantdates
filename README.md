# 📚 Exam Resource Hub
### Competitive Exam Study Portal  
### By Arnav Batra  

A **minimal, student-first exam resource website** built for **fast revision and one-click access to essential competitive exam resources**.

Designed especially for aspirants preparing for:

- UPSC
- SSC
- Banking
- CDS
- NDA
- CUET
- Railways
- State PCS
- School GK Olympiads

This project has evolved from a simple **Important Days revision tool** into a **centralized exam resource terminal**, bringing multiple high-value static resources into one clean interface.

---

## ✨ Features
- 📅 **Important Dates & Days**
- 🌍 **World Organizations & Headquarters**
- 🏛️ **Parliaments of Countries**
- 📚 **Master Resource Hub (All-in-one)**
- 🔗 Quick access to external study resources
- 📱 Fully responsive for mobile + desktop
- ⚡ Super fast static deployment via GitHub Pages
- 🎯 Built specifically for revision speed
- 🧠 Designed for competitive retention & last-minute prep
